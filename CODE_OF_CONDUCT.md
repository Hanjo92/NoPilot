# Code of Conduct

## Our Pledge

We want NoPilot to be a welcoming, respectful, and constructive community for everyone.

Contributors, maintainers, and participants in this project are expected to make participation harassment-free for all people, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- being respectful and considerate in discussions
- giving and receiving constructive feedback
- focusing on what is best for the project and community
- showing empathy toward other community members

Examples of unacceptable behavior include:

- harassment, insulting language, or personal attacks
- trolling, deliberate intimidation, or derogatory comments
- publishing others' private information without explicit permission
- any conduct that would be inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing these standards and may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is representing the project or its community.

## Reporting

If you experience or witness unacceptable behavior, please open a GitHub issue with minimal detail and ask for a private follow-up if needed. Security-sensitive issues should follow the process in `SECURITY.md`.
