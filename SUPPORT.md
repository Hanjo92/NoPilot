# Support

If you run into a NoPilot bug, setup issue, provider problem, or packaging problem, please open an issue in the project repository:

https://github.com/Hanjo92/NoPilot/issues

Helpful details to include:

- NoPilot version
- VS Code version
- Operating system
- Active provider and model
- Whether you used the Activity Bar menu, status bar picker, settings dashboard, or Command Palette
- Any relevant logs from **Output > NoPilot**

For usage, provider setup, remote Ollama, OpenAI-compatible endpoints, settings, and data handling, check the main README in the same repository.
